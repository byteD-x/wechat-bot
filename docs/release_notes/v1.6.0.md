# v1.6.0 更新内容

## Features

- 新增“微信聊天导出中心”页面，支持按步骤完成账号探测、解密、联系人筛选导出和一键应用，降低首次使用门槛。
- 新增导出中心后端服务与 API，支持解密任务状态查询、联系人读取和导出后配置预览，为运营与知识库准备提供统一入口。

## Improvements

- 模型中心默认模型显示逻辑优化：未绑定认证档案时优先展示用户已保存的默认模型，减少“保存成功但界面像回退”的误判。
- 模型中心 provider 合并策略优化：优先使用后端实时 overview 数据，避免被静态 catalog 默认值覆盖。
- 主进程与渲染层的页面装配链路补齐导出中心入口，模型页/设置页间的跳转与状态同步更一致。

## Fixes

- 修复 OpenAI 等 provider 在 `presets` 为空场景下保存默认模型后被根配置回填覆盖的问题。
- 修复模型中心切换默认模型时选中认证档案未同步更新的边界问题，降低保存后运行时模型不一致风险。
- 修复若干 IPC 安全边界与前端交互回归问题，减少异常输入导致的误触发。

## Performance

- 优化模型认证中心相关状态刷新与测试覆盖，减少重复重算与前端重复渲染带来的无效开销。
