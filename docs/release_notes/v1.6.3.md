# v1.6.3 更新内容

## Features

- 新增模型侧 Tool Calling 安全工具流，默认关闭，开启后仅暴露受控只读和预览工具，并沿用现有工具注册、权限和 trace 边界。
- 新增只读 MCP adapter，可通过本机 JSON-RPC 调用白名单摘要工具，不开放 Prompt 预览、任意命令、文件写入或动态插件。
- 新增知识库治理闭环，支持本机 API、设置页、CLI、批量预览、批量写入、批量重建、后台任务、索引摘要和固定 inbox 预览后受控入队。
- 新增默认关闭的语义响应缓存与内存级脱敏 trace 摘要，便于在不记录聊天正文、Prompt、token 或完整路径的前提下排查运行时行为。
- 新增 Web API 容器部署切片，明确容器只覆盖 Web API、readiness 和离线评测，不承诺微信桌面自动化容器化。

## Improvements

- 优化 RAG 检索链路，支持混合检索、查询改写和引用绑定评测，提升知识召回与可验证回答的一致性。
- 优化受控工具工作流，支持规划、反思和最多一次 schema-safe repair，并扩展维护 dry-run 白名单工具。
- 优化知识库治理安全边界，所有预览、队列、索引和固定 inbox 响应只返回脱敏摘要，不返回正文、chunk text、embedding 或完整本机路径。
- 更新 API 路由索引和治理文档，减少接口清单漂移并明确配置、状态字段和部署边界。

## Fixes

- 修复知识库重建入口的 IPC 放行问题，确保设置页受控重建可以调用对应本机 API。
- 修复 CI 中回复策略局部变量未使用导致的静态检查失败。
