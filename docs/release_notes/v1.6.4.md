# v1.6.4 更新内容

## Improvements

- 新增未签名社区版发布通道，个人开发者可在没有代码签名证书时发布明确标识的 Windows 安装包。
- 增强 Windows 发布门禁，默认官方签名版仍要求 Authenticode 通过，社区版只在显式选择后放宽签名检查。
- GitHub Release 现在会同步 `latest.yml` 和 setup `.blockmap`，保留后续自动更新元数据通道。
- 手动发布脚本会在未签名社区版标题和正文中标明风险，提示用户使用 `SHA256SUMS.txt` 校验下载文件。

## Fixes

- 修复发布资产清理会误删 `latest.yml` 和 `.blockmap` 的问题，避免构建后缺少更新元数据。
