# v1.1.1 更新内容

## Features

- 新增成本管理页面，支持按会话查看模型、Token 和费用明细。
- 新增运行期记忆增强和聊天记录风格学习，回复可以参考更多上下文和历史表达方式。
- 新增启动进度、运行诊断和一键恢复入口，排查启动问题更直接。
- 新增健康监控和提示词预览，配置前可以先确认系统提示词效果。
- 新增 Windows 安装版和便携版发布产物，下载安装与分发更方便。

## Improvements

- 优化设置、消息和日志页面，支持分模块保存配置、查看联系人画像和编辑专属 Prompt。
- 调整回复链路和状态反馈，回复过程中的运行信息更完整。

## Fixes

- 修复桌面端导航、按钮交互、配置保存和语音转文字等主链路问题。

## Performance

- 优化模型请求和上下文处理的重复开销，长时间运行时更稳定。
