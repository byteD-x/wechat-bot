# v1.5.0 更新内容

## Features

- 新增统一的模型认证中心体验，可在同一处管理 OpenAI、Claude、Gemini、Qwen、Kimi、豆包和元宝等服务的可用模型与登录方式。
- 新增本地授权后台同步进度展示，模型页和设置页在首次打开时会自动补刷新，不需要手动反复重载才能看到最新状态。

## Improvements

- 调整模型与设置页面的交互逻辑，切换页面或切换服务卡片后会尊重当前用户选择，不再因为后台初始化或慢响应打断操作。
- 更新默认模型目录与认证适配能力，补齐多家供应商的最新主力模型和更完整的运行时配置默认值。

## Fixes

- 修复应用刚启动时切换页面后又被拉回仪表盘的问题。
- 修复模型保存后晚到响应会把用户拉回旧卡片、设置页后台重复拉取配置导致界面卡顿的问题。
- 修复多种本地授权与 OpenAI 兼容运行时的状态识别、回退与健康检查不一致问题。

## Performance

- 优化模型目录、认证注册表和本地授权同步链路，模型中心概览与配置接口由秒级阻塞下降到毫秒级返回。
- 将本地授权扫描改为后台异步执行，并复用缓存快照，减少冷启动期间的界面等待和重复请求。
